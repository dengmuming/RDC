# RDC

#### 介绍
RDC 是基于.Net Framework 4.7.2 .Net 6 框架的 C# WinForm Redis 客户端。
UI 采用了SunnyUI
NuGet扩展包：Newtonsoft、StackExchange.Redis。


#### 安装教程
